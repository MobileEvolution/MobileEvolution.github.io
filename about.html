Mobile Evolution Dataset

About The Dataset

Understanding how software has been created is helpful for developing better software. Unfortunately, mobile research is often inhibited from a lack of evolutionary mobile development data. To address this issue, we present a large dataset of the evolutionary history of open-source Android apps to assist researchers in better understanding the progression of these apps, their development and current state. Our dataset contains the evolutionary history of over 2,000 Android apps including data on the app's GitHub Issues, permissions changes, general change ownership and release details. This work provides researchers with a rich set of evolutionary data that they can use in their own work.
